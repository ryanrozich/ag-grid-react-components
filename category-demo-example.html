Features:

  • Dark theme optimized colors with semi-transparent backgrounds
  • Smooth hover effects with scale transformation
  • Consistent pill shape with rounded borders
  • High contrast text for readability
  • Unique color for each category type
  • Lightweight and performant

Usage in AG Grid:

{
  field: "category",
  headerName: "Category",
  width: 130,
  cellRenderer: CategoryCellRenderer,
  autoHeight: true,
}
